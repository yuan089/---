1.基礎模組-音樂播放機
物品列表
-隨身聽
-特製jukebox
-客製化卡帶合集
-刻錄機
2.特性
-一切附加音樂將會由DLC形式發佈，基礎模組爲靈活的音樂播放器
-隨身聽可以隨身攜帶並播放，附近玩家可以聽見小範圍聲音
-特製jukebox可以播放任何碟片，並一直重複播放下去，parrot可以與之互動，播放時有特效
-客製化卡帶合集可以裝載60首音樂，你可以在刻錄機刻錄音樂ID於其中
-刻錄機可以刻錄歌曲，其中有三種選項，刻錄身上所有的歌曲，檢索或選擇刻錄歌曲，一鍵清除所有歌曲
3.合成
-隨身聽合成由一銅錠一鐵錠合成
-特製jukebox由遊戲自帶的jukebox合成
-客製化卡帶合集由一個紅石粉末一個鐵錠合成
-刻錄機由九個銅錠合成
